# Moble Planejados — Design Reference

## Design Analysis (Original Site)

### Visual Identity
- **Brand**: Moble Planejados — Marcenaria de Alto Padrão
- **Tagline**: "A forma segue a precisão" (Form follows precision)
- **Location**: São Paulo, BR
- **Specialty**: High-end carpentry with technical rigor

### Color Palette
- **Primary Background**: White/Off-white (clean, minimal)
- **Text Primary**: Very dark gray/black (#1a1a1a or similar)
- **Accent/Dark Sections**: Deep charcoal/black (#1f1f1f or #000)
- **Accent Color**: Warm brass/gold tones (visible in hardware details)
- **Borders**: Light gray (#e5e5e5 or similar)

### Typography
- **Headline Font**: Bold, geometric sans-serif (appears to be a modern sans)
- **Body Font**: Clean, readable sans-serif
- **Style**: Minimalist, technical, professional
- **Hierarchy**: Large headlines with italic secondary text ("a precisão" in italics)

### Layout & Structure
1. **Header/Navigation**: Sticky top nav with logo + menu items (Ambientes, Processo, Portfólio, Solicitar Orçamento)
2. **Hero Section**: Split layout — text on left, large hero image on right
3. **Technical Detail Badge**: Small card with "Detalhe Técnico" and "Sistema Soft-Close 45º"
4. **Category Tabs**: Dark background section with 6 numbered categories (01-06)
5. **Portfolio Section**: Project showcase with images and specifications
6. **Manifesto Section**: Three-column layout with numbered steps (01, 02, 03)
7. **CTA Section**: Call-to-action with WhatsApp and Email buttons
8. **Footer**: Dark background with links and social media

### Key Visual Elements
- Large, high-quality product photography (kitchens, closets, details)
- Minimalist spacing and breathing room
- Technical specifications displayed in clean cards
- Numbered sections (01, 02, 03, etc.) for visual structure
- Dark sections contrast with white sections
- No excessive decoration — pure functional design

### Interaction Patterns
- Smooth scrolling navigation
- Hover states on links and buttons
- Category tabs for portfolio filtering
- Direct contact CTAs (WhatsApp, Email)

### Design Philosophy
**Precision + Minimalism**: Every element serves a purpose. No decoration for decoration's sake. Technical accuracy reflected in visual design. Silence and clarity are the aesthetic.

## Design Direction (Chosen)

**Theme**: Minimalist Technical Elegance

**Core Principles**:
- Precision in every pixel
- Whitespace as a design tool
- Technical specifications visible but not intrusive
- High-quality imagery as hero element
- Dark/light contrast for visual hierarchy

**Color Philosophy**:
- White as primary background (cleanliness, precision)
- Deep charcoal/black for contrast and authority
- Warm brass accents (technical hardware reference)
- Gray tones for secondary information

**Typography System**:
- Bold geometric sans for headlines (technical feel)
- Clean sans for body text (readability)
- Italic for secondary/poetic text (design sophistication)

**Signature Elements**:
- Numbered sections (01, 02, 03)
- Technical specification cards
- Split hero layout (text + image)
- Dark section breaks

**Brand Essence**:
High-end carpentry for discerning clients who value precision, quality, and silence. Technical mastery expressed through minimalist design.

**Personality**: Professional, precise, sophisticated, trustworthy.
